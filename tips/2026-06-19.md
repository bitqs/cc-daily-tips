# 2026-06-19 | 给 Claude Code 接上外部工具：3 分钟配置一个 MCP Server

## 技巧

用 `claude mcp add` 命令给 Claude Code 接入 MCP Server，让它直接操作数据库、调 GitHub API、读写任意目录——不需要改一行代码，装完即用。

## 怎么做

**1. 查看当前已配置的 MCP Server**

```bash
claude mcp list
```

**2. 接入文件系统 Server（让 Claude 能读写指定目录）**

```bash
# 全局生效（所有项目可用）
claude mcp add filesystem --scope global -- \
  npx -y @modelcontextprotocol/server-filesystem ~/projects

# 或只对当前项目生效（写入 .claude/settings.json）
claude mcp add filesystem -- \
  npx -y @modelcontextprotocol/server-filesystem .
```

**3. 接入 GitHub Server（让 Claude 直接查 PR、Issues）**

```bash
# 先设置环境变量（也可以写进 shell 配置文件）
export GITHUB_PERSONAL_ACCESS_TOKEN=ghp_xxx

claude mcp add github --scope global -- \
  npx -y @modelcontextprotocol/server-github
```

配好后，在 Claude Code 里直接问："帮我列出这个仓库最近未关闭的 Issues"——它会通过 MCP 直接调 GitHub API，不需要你去网页复制。

**4. 接入 Postgres 数据库**

```bash
claude mcp add postgres --scope global -- \
  npx -y @modelcontextprotocol/server-postgres \
  postgresql://user:pass@localhost/mydb
```

之后 Claude Code 可以直接执行 SQL 查询，帮你排查数据问题。

**5. 验证 Server 是否接入成功**

```bash
claude mcp list
# 输出示例：
# filesystem  stdio  npx -y @modelcontextprotocol/server-filesystem ~/projects
# github      stdio  npx -y @modelcontextprotocol/server-github
```

**6. 删除某个 Server**

```bash
claude mcp remove filesystem
```

**作用范围说明**

| 命令参数 | 配置文件位置 | 生效范围 |
|---|---|---|
| `--scope global`（默认）| `~/.claude/settings.json` | 所有项目 |
| `--scope local` | `.claude/settings.json` | 当前项目 |
| `--scope project` | `.claude/settings.local.json` | 当前项目（不提交 Git）|

## 为什么

Claude Code 内置工具只能读写本地文件、执行 Shell 命令。但实际开发中你需要的远不止这些：查 GitHub Issue、查数据库、调内部 API……每次都要自己查完再粘贴给 Claude，效率极低。

MCP（Model Context Protocol）是 Anthropic 推的开放标准，相当于给 AI 的「插件协议」：

- **装上 GitHub Server**：Claude Code 能直接拉 PR diff、查 Issues、评论——不用切浏览器
- **装上 Postgres Server**：Claude Code 能直接查表结构、执行 SQL，排查线上数据问题
- **装上 Filesystem Server**：Claude Code 能访问项目目录以外的路径，适合 monorepo 场景

`--scope global` 配置一次，所有项目终身受益；`--scope local` 配置提交进 Git，团队成员 clone 后直接可用（Server 本身不含敏感信息，只是命令路径）。

社区已有几十个开源 MCP Server：[modelcontextprotocol/servers](https://github.com/modelcontextprotocol/servers)，找到适合的直接 `claude mcp add` 即可。
